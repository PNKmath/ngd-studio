# NGD 레이아웃/서식 규칙

양식지에서 추출한 레이아웃 및 서식 규칙. Human-only 조작(키보드 단축키, GUI 조작) 제거됨.

## 페이지/단 배치

- 고등부 시험지: **한 단에 2문항** 배치 원칙
- 중등부 시험지: 한 단에 3문항
- 2단 레이아웃, 한 쪽에 4문제 (좌2 + 우2)

## 머리말

- 기출작업양식의 머리말 오른쪽 칸에 **학년/과정/과목/출판사와 시험 범위** 기재
- 고1학년인 경우 학년/과목/출판사와 범위를 기재
- 출판사 모를 때는 입력하지 않음

## 서체/스타일

- 서체: **나눔고딕 10**, 수식크기 **11**, 수식서체 **HYhwpEQ**
- F6 스타일: **바탕글 1개만** (다른 스타일 없어야 함)
- `styleIDRef="0"` (바탕글) 하나만 사용

## 미주와 문제

- 미주번호와 문제 텍스트 사이는 **붙여 쓴다** (한 칸 띄어쓰기 금지)
- endNote 마커 직후 바로 문제 텍스트 시작

## 문제↔보기 간격

- 문제와 객관식 보기 사이에 **한 줄 띄움** (빈 문단 1개)
- 문제와 보기 사이에 [그림], [조건], `< 보 기 >` 가 있는 경우에도 **한 줄 띄움**

## 문제↔배점

- 문제 텍스트와 배점 사이는 **한 칸** 띄움
- 배점 숫자는 **반드시 수식**(`<hp:equation>`)으로 입력
- 형식: `[` + 수식(배점) + `점]`

## 배점 줄넘김

- 배점이 문제 뒤에 잘려서 다음 줄로 넘어가면:
  - 다음 줄로 내린 후 **오른쪽 정렬** (`paraPrIDRef="4"`)

## 선지 간격

- 보기 ①②③④⑤ 간격: **탭키 3번** (`<hp:tab>` 3개)
- 선지 값이 긴 경우: 각 선지를 개별 문단으로 배치
- 특수 보기 양식(p:q 형태 등)은 양식지 4~5p 템플릿 참조

## 독립줄 수식

- 문제에서 수식이 독립줄(별도 줄)에 올 때: **tab으로 들여쓰기** 후 작업
- **해설에서는 적용하지 않음**

## 그림 배치

### 오른쪽 배치
- 그림이 문제 오른쪽에 위치할 경우:
  - 문제 **첫 어절 뒤**에 배치
  - `textWrap="AROUND"` + `horzAlign="RIGHT"` + `vertAlign="TOP"` 설정
  - ('큰 그림, 오른쪽 정렬, 위쪽 맞춤')

### 단독 배치 (중간/끝)
- 그림이 문제 중간이나 끝에 단독으로 위치할 경우:
  - `treatAsChar="1"` (글자처럼 취급)
  - `paraPrIDRef="2"` (가운데 정렬)

## (가), (나), (다) 표기

- 문자표 사용하지 않고 **한글 그대로** 입력: `(가)`, `(나)`, `(다)`

## 박스 종류

양식지에 다음 5가지 박스 템플릿이 정의되어 있다:
1. **빈박스** (empty_box) — 답안 작성 공간
2. **보기틀** (bogi) — `< 보 기 >` 헤더 + ㄱ/ㄴ/ㄷ 항목
3. **그림보기틀** (image_choice) — 그림 포함 선지
4. **조건틀** (condition) — (가)/(나)/(다) 조건
5. **증명틀** (proof) — `[ 증 명 ]` 헤더 + 증명 내용

## 박스 내부 정렬

- 박스 안에 (가)/(나)/(다) 등의 글이 있을 때:
  - 각 항목 뒤 텍스트가 자연스럽게 들여쓰기되도록 정렬

## 제곱근표/상용로그표

- `[제곱근표]`나 `[상용로그표]`는 **표를 직접 작성**하여 입력
- 이미지 붙여넣기 금지
