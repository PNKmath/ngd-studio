## 6. builder 영향

RENAMED·CHANGED 항목이 기존 시험지의 `[중단원]` 태그에 미치는 영향:

| 변경 항목 | 태그 표기 변화 | 영향 범위 |
|---|---|---|
| 수1.G.name `지수로그` | `[지수함수와 로그함수]` → `[지수로그]` | 향후 신규 작업부터 적용. 기존 완성 HWPX는 변경 없음 (이력 보존) |
| 미적.N.topics[1] `여러 가지 미분법` | 주제 태그가 해당 값인 경우에만 영향 | 향후 신규 작업부터 적용 |
| 기하.U.name `공간도형` | `[공간도형과 공간좌표]` → `[공간도형]` | 향후 신규 작업부터 적용. 기존 완성 HWPX는 변경 없음 (이력 보존) |

과거 작업 결과물의 [중단원] 태그는 옛 표기 그대로 유지하는 것이 원칙이다.
