# NGD 오검 체크리스트

## 편집오검 내역표 22개 고정 항목

HWPX 문서 끝에 있는 편집오검 내역표의 고정 항목. 오검 시 해당 항목에 문제번호를 기입한다.

| 번호 | 내역표 항목 | 검증 내용 |
|------|-----------|----------|
| 1 | 배점 위치 수정 | 문제와 배점 사이 한 칸, 배점 수식 사용 |
| 2 | 단서 조항 위치 수정 | (단, ~) 문제 끝에서 한 칸 띄우고 입력 |
| 3 | 조건틀, 보기틀, 증명틀 수정 | 조건/보기/증명 틀 형식 |
| 4 | 확률과통계, 좌표 로마체 수정 | 순열/조합/확률/분포 기호 rm체, 단위/도형 대문자 rm체 |
| 5 | therefore / because 수정 | therefore/because 뒤 `~` |
| 6 | cdots 수정 | `[수식]+` `` `cdots` `` `+[수식]` |
| 7 | 괄호 수정 | 분수 괄호 `left(` `right)` |
| 8 | 소문항 번호 (1), (2) 로 수정 | 소문항 번호 형식 |
| 9 | 통수식 수정 | 등호 단위로 끊기 |
| 10 | 워딩규칙, 중단원표페이지 삭제 | 워딩 규칙 준수, 불필요 페이지 삭제 |
| 11 | 제작연월일 수정 | 제작연월일 확인 |
| 12 | 글자크기, 수식 크기 수정 | 나눔고딕 10, 수식크기 11 |
| 13 | 비표 수정 | 비표(제곱근표/상용로그표) 한글 표 작업 |
| 14 | 바탕글 이외의 스타일 삭제 | F6 스타일 바탕글 1개만 |
| 15 | 문제의 독립수식 tab으로 수정 | 독립줄 수식 탭 들여쓰기 (해설 제외) |
| 16 | 중단원명 수정 | 중단원/난이도 입력 확인 |
| 17 | 콤마 수정 | 쉼표 뒤 한 칸 `~` 또는 `` ` `` |
| 18 | 그림, 표 위치 수정 | 그림 오른쪽/단독 위치 정렬 |
| 19 | 문제 또는 그림과 객관식 보기 사이에 한 줄 띄움 | 문제와 선지 사이 Enter |
| 20 | 미주와 문제 시작시 한 칸 띄움 수정 | 미주와 문제사이 띄어쓰기 없음 |
| 21 | 서술형문제번호는 숫자로 수정 | `[서술형 N]` 형식 |
| 22 | 해설 항상 좌측 정렬(들여쓰기 안함) | 해설 좌측 정렬 |

## 추가 검증 항목 (내역표에 없는 것)

이 항목들은 22개 고정 항목에 해당하지 않는 추가 검증 사항이다. 위반 발견 시 **두 번째 편집오검 내역표**에 기재한다.

- [ ] 정답 볼드 처리 금지
- [ ] 정답라인 밑 해설 문구 삭제
- [ ] 다른 풀이 있으면 `[다른 풀이]` 입력
- [ ] shift+enter 사용금지 (정답 라인 2줄 넘어갈 때만)
- [ ] 절댓값: 수식 기호 사용
- [ ] 내적: `cdot` (bullet 아님)
- [ ] 부등호 뒤 음수: `it` 사용
- [ ] 극한 음수: `lim_{x->it-2}`
- [ ] 수식창 한글 작업 안함
- [ ] 여집합: 소문자 `c`, 합집합/교집합: `cup`, `cap`
- [ ] 명제: `p` `` ` `` `->` `` ` `` `q`
- [ ] cases: `&` 3개 정렬
- [ ] 삼각함수/로그 분수 뒤: 올바른 입력
- [ ] (가)(나)(다): 문자표 사용 안함
- [ ] 머릿말 오른쪽칸: 학년/과정/과목/출판사/시험범위
- [ ] 서술형 전체 배점은 소문항에 점수 있으면 적지 않음
