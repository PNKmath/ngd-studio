# PDF 자동 크롭 작업 전용

이 폴더는 PDF 시험지에서 각 문제를 개별 이미지로 크롭하는 전용 작업 폴더다.
시험지 제작, 오검, HWPX 조립 등 다른 작업은 수행하지 않는다.

## 도구

- **PyMuPDF (fitz)**: PDF → 페이지 이미지 변환 (200dpi)
- **Pillow (PIL)**: 이미지 크롭
- **Read 도구**: 페이지 이미지 분석 (문제 영역 감지)

## 시험지 구조

- 고등부 시험지: 2단 레이아웃 (좌/우)
- 한 페이지에 보통 2~4문제
- 문제 번호: 미주 번호 (1, 2, 3...) 또는 원문자 (①②③)
- 서술형: `[서술형 N]` 패턴
- 앞부분 = 문제 페이지, 뒷부분 = 해설 페이지
- 해설 페이지는 크롭하지 않음 (문제 페이지만 처리)

## 해설 페이지 판별

- `[정답]`, `해설`, `풀이` 텍스트가 주로 보이면 해설 페이지
- 해설 페이지의 문제 번호와 혼동하지 않을 것

## 출력

- 크롭된 이미지: `q01.png`, `q02.png`, ...
- 결과 JSON: `crop_results.json`
